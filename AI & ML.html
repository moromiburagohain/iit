Syllabus

Detailed module breakdown  expand to view topics and resources.

Module 101 Mathematics and Statistics: Delve into Linear Algebra, Multivariate Calculus, Statistics, and Probability. Learn Linear Regression, Outlier Rejection, and both Unconstrained and Constrained Optimization, including Stochastic Search with GA and PSO.
Module 102 Python Programming: Master Python Fundamentals, String Objects and Collections (Tuples, Set, Dictionaries, Functions). Get a thorough introduction to Pandas, covering basics, Series, Dataframes, text data, indexing, selecting, visualization, and critical data manipulation and cleansing techniques like handling missing values and skewness.

Module 201 Feature Space and Dimensionality Reduction, Unsupervised Learning, Generative Models, Pattern Mining: Gain proficiency with NumPy, SciPy, Scikit- learn, Seaborn, and Matplotlib. Understand the Machine Learning Task Workflow, Feature Space transformations, and separability. Explore Hashing Techniques for Feature Space Partition, Feature Subset Selection, Principal Component Analysis (PCA), Linear Discriminant Analysis (LDA), and Exploratory Factor Analysis (EFA).
Module 202 Unsupervised Learning, Generative Models, Pattern Mining: Explore various Unsupervised Learning techniques including Adaptive Thresholding, K-Means (and its variants), Hierarchical Agglomerative Clustering, DBSCAN, Nonparametric Estimation, and Mean-Shift Clustering. Learn Performance Analysis of Clustering Algorithms, Parametric Estimation (Bias, Variance, MLE), Learning Mixture Models, Unsupervised Representation Learning, Probabilistic Latent Semantic Model (PLSA), Association Rule Mining, Suffix Tree and Sequence Mining, Markov Process, Hidden Markov Model (HMM), and Prediction with Autoregressive Process Models.

Module 301 Supervised Learning- Classification and Regression Dive into Supervised Learning with Bayesian Classification, Discriminant Functions, Logistic Regression, Perceptron (Binary and Multicategory), and Multilayer Perceptron (Architecture, Backpropagation, Training Issues). Master K-NN, Radial Basis Function ANN, Data Balancing Techniques, Classification and Regression Trees, Neural Trees, Ensemble Methods (Bagging - Random Forest, Boosting - AdaBoost), and Support Vector Machines (SVM).
Module 302 Deep Learning and Computer Vision: Get a comprehensive Introduction to Deep Learning, covering Convolutional Neural Networks (Popular Architectures, Learning Algorithms, Training Issues), Recurrent Neural Networks (RNN) and LSTM, Encoder-Decoder Networks, and Generative Adversarial Networks. Gain practical Computer Vision skills using the OpenCV Library for Feature Extraction, Object Detection, Face Detection/ Recognition/Search, Object Tracking, and Image Segmentation in DL Frameworks.

Module 401 Natural Language Processing, Speech and Audio Processing: Explore Natural Language Processing with the NLTK Library, including Text Normalization (Tokenization, Lemmatization, Morphology), n- gram Language Models, Vector Semantics (Word2Vec, GloVE), Neural Models, Neural Language Models, and Sequence Labeling Tasks (CNN and RNN Models for NLP). Learn Speech and Audio Processing using LibRosa for basic operations, feature extraction for classification/synthesis, and building Speech Recognition and Synthesis Systems.
Module 402 Machine Learning on Cloud, Counselling and Final Project: Understand Cloud fundamentals and ML tools on leading platforms (AWS, Azure, GCP), including how to deploy ML projects on the cloud. Benefit from Career Counselling and Interview Facing Tips & Tricks, culminating in a Capstone Project and Final Project Presentation to showcase your acquired skills.
