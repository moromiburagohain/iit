About the Course

This Professional Certification Programme in Cyber Security covers foundational to advanced topics such as network security, ethical hacking, cloud security, cryptography, incident response and forensics.

Who should apply?

Graduates, IT professionals, aspiring security engineers.

Outcome

Industry-ready skills, capstone project, career guidance.

Program Highlights
  • Real-world labs & capstone – simulated attack and defense.
  • Tools-focused – Kali, Metasploit, Burp, Nmap, Splunk, ELK.
  • Career support – resume, interview prep, certification guidance.

Learning Path – 12 Months

A progressive journey from foundations to advanced operations, aligned to industry needs. Summary adapted from the course brochure.

Months 1–2 – Foundation

Intro to security, networks, OS basics, Python/scripting fundamentals.

Months 3–4 – Core Concepts

Network security, cryptography, web application and secure coding basics.

Months 5–6 – Offensive Security

Ethical hacking, penetration testing, vulnerability assessment (Nmap, Metasploit).

Months 7–8 – Defensive Security

SIEM & SOC fundamentals (Splunk/ELK), malware analysis, endpoint defenses.

Months 9–10 – Advanced Modules

Cloud security, governance, IAM, risk & compliance.

Months 11–12 – Incident Response & Capstone

Digital forensics, incident response, capstone simulation, career prep.

Hands-On & Tools

Extensive practical exposure – Kali Linux, Burp Suite, Nmap, Metasploit, Wireshark, Splunk, ELK; plus cloud security tooling (AWS/Azure/GCP best practices).

Activities
  • Live red-team / blue-team exercises
  • Capture The Flag (CTF) events
  • Weekly lab assignments & code reviews
  • Group projects and peer code audits

Syllabus – Overview

  • Introduction to Cybersecurity & Threat Landscape
  • Computer Networks, OS, Security Basics
  • Basics of Programming & Scripting (Python)

  • Network Security & Secure Design
  • Cryptography & Data Security
  • Web App Security, Secure Coding
  • Ethical Hacking, Vulnerability Assessment & Exploitation

  • SIEM, SOC Operations (Splunk/ELK)
  • Malware Analysis, IDS/IPS
  • Cloud Security, IAM, Governance & Risk

  • Incident Response & Digital Forensics
  • Capstone: Simulated Cyber Attack & Defense
  • Resume & Interview Preparation; Certification Prep

Eligibility & Pre-requisites

Eligibility
  • Graduate degree (any discipline; CS/IT/Engineering preferred).
  • Basic knowledge of computer networks and operating systems.
  • Familiarity with programming fundamentals (Python, C, Java – helpful but not mandatory).
  • Working professionals with 1–2 years experience recommended but not mandatory.
Pre-requisites

No prior cybersecurity experience required – course starts with fundamentals and progresses to advanced topics.
